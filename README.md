### Instructions

```bash
$ git clone https://github.com/kartikreddy18/contact.git
$ cd contact
$ pnpm install
$ pnpm dev
```

### Tech Stack
  - Next.js 
  - Redux
  - Tailwind
  - Typescript

## Api used
  - <https://disease.sh/v3/covid-19/all>
  - <https://disease.sh/v3/covid-19/historical/all?lastdays=all>
  - <https://disease.sh/v3/covid-19/countries>
